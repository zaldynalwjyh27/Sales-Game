## About the Project

JISR (جسر) is an interactive sales simulation game that trains sales skills through realistic sales scenarios. The application allows users to practice selling techniques, discover customer pain points, handle objections, and close deals in a simulated environment with AI-powered clients.

Key features include:
- Multi-role simulation (Seller, Client, Evaluator)
- Real-time communication using Pusher
- AI-powered client simulation using OpenAI
- Arabic language interface with RTL support
- 12 different sales scenarios with various personas
- Comprehensive evaluation system
- Round-based gameplay with scenario rotation
